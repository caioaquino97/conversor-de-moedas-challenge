# Conversor de Moedas - Challenge ONE 

Este projeto é um conversor de moedas interativo desenvolvido em Java, como parte do desafio do programa Oracle Next Education (ONE).

## 📋 Funcionalidades
- Conversão de moedas em tempo real.
- Interface prática via terminal (console).
- Suporte para: Dólar (USD), Real (BRL), Peso Argentino (ARS) e Peso Colombiano (COP).
- Registro de consultas em arquivo JSON utilizando a biblioteca Gson.

## 🛠️ Requisitos e Configuração
1. **Java JDK 17** ou superior.
2. **Biblioteca Gson:** Necessária para o processamento dos dados JSON.
3. **Chave de API:** - Obtenha uma chave gratuita em: [exchangerate-api.com](https://www.exchangerate-api.com/)
   - No código, substitua o campo da API Key pela sua chave pessoal.

## 🚀 Como usar
1. Clone este repositório.
2. Importe o projeto na sua IDE (IntelliJ, Eclipse, etc).
3. Execute a classe `Principal.java`.
4. Escolha a opção de conversão no menu e digite o valor desejado.
